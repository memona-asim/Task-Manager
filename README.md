# Basic Task Manager (Angular)

This is a simple Task Manager application built using Angular to learn core Angular concepts such as components, services, routing, and data binding.

---

## Features
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- View all tasks in a clean UI  

---

## Tech Stack
- Angular
- TypeScript
- HTML / CSS

---

## Run Locally

### Prerequisites
Make sure you have **Node.js** and **Angular CLI** installed.

```bash
npm install -g @angular/cli
